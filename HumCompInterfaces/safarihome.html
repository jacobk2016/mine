Safari Lessons

Welcome to The Safari Lessons!


Safari is the internet browser that comes installed with an Apple operating system, (Mac). Safari has been around for a long time; Don't worry, even if you have an older operating system, Safari is probably installed on your Apple computer.


Safari follows the same basic format as other internet browsers. Check out the diagram below. When you're ready to see the descriptions of the labeled parts, scroll down. Some of the labels are very similar to the ones in the Internet Browser Basics tutorial, so be sure to check that lesson out if you need more information about that particular part.



1. Back and Forward Buttons

You can learn more about this in the interface section of Internet Browser Basics.


2. Tabs

You can learn more about this in the interface section of Internet Browser Basics.


3. Address Bar

You can learn more about this in the interface section of Internet Browser Basics.


4. Search Bar

This is where you can type in what you would like to search for on the internet. If you type something here and press "Enter" on your keyboard, your browser will automatically perform a google search with your text.

5. Top Right Buttons

If you keep scrolling down, we've provided a little more detailed information about what this area does.




1. Your Minimize, Maximize/Restore and Close buttons

You can learn more about this in the interface section of Internet Browser Basics.


2. Current Page Menu (Left) and Settings (Right)

If you click on the Current Page Menu button (the one on the left that kind of looks like a piece of paper with a folded corner) it will give you some options about what you can do with your current page. You can open new tabs or windows, view the source code of the webpage, etc.

If you click on the Settings button (the one on the right that looks like a gear) you'll see a menu of items you can change or set up for your web browser. This is where you'll find your security settings, browsing history and other things.

Now that you know how all the basic parts of Safari work, you are ready to try updating your Safari Security Settings. Click "Next Lesson" below when you are ready to move on to the security lesson.
